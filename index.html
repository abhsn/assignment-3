The basic differnece between flexbox and grid is that flexbox was desinged for one dimensional layout, either a row or column can be handled at same time. On the other hand, grid was desinged for two dimensional layout, both row and column can be handled at the same time. Say, you have five divs under a div called wrapper using flexbox with wrap value in flex-wrap property and your wrapper divs width is 600px. Each div under the wrapper is 200px by width. Now first three divs will be in first row containing 200px each. But, last two divs will take full available width without respecting divs specified width. So, fix this problem you have to either use two rows in flexbox or you can use grid with grid-template-columns: repeat(3, 1fr); declatation.

Both tailwind and bootstrap are CSS framework used for building website without writing CSS from scratch. Tailwind is a low-level CSS framework, means it gives more power to make website more flexible. Bootstrap is a component based framework that helps to develop the website quickly. Tailwind offers predesigned widgets to build website from scratch. On the other hand, bootstrap comes withpre-styled components like, navigation bar, buttons, responsiveness etc.

When laying out a document, the browser's rendering engine represents each element as a rectangular box according to the standard CSS basic box model. CSS determines the size, position, and properties (color, background, border size, etc.) of these boxes. Every box is composed of four parts (or areas), defined by their respective edges: the content edge, padding edge, border edge, and margin edge.

Semantic HTML tags are those that clearly describe their meaning in a human and machine readable way. Tags such as <header>, <footer> and <article> are all considered semantic because they accurately describe the purpose of the element and the type of content that is inside them. Non-semantic tag like <div> is not machine readable, screen reader, web crawler can't read those tags. Hence to fix that problem semantic tag was introduced.
